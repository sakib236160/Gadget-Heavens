## How to Install & Run
* Clone the Repository:
     git clone https://github.com/sakib236160/Gadget-Heavens.git
* Navigate to the Project Directory:
    cd Gadget-Heavens
* Install Dependencies:
    npm install
* Start the Application:
    npm run dev
